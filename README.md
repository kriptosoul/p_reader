# Pebble TXT Reader

Simple TXT reader for Pebble Time (SDK 3).

Features:
- Load a TXT file from resources.
- Scroll with Up/Down buttons.
- Settings menu (Select) to change font size and theme.
- Persist reading position and settings between runs.

Build:
- Use the Pebble SDK (pebble-sdk-3). From project root run `pebble build`.
- This will generate a `.pbw` you can sideload or upload.

Notes:
- The included `resources/data/book.txt` is a short sample. Replace it with your own TXT.
